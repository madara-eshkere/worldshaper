# Спека: Механики, Скриптовые триггеры, Эскалации (M1)

Статус: v1, 2026-07-18. Реализует ADR-0014 (Механика = данные) и ADR-0009/0011
(Рефлексы: важность решается триггерами/счётчиками, без Наблюдателя-LLM).
Это формат, который на M2 будет **генерировать Режиссёр** (как данные, не код).

## Механика — данные, не код

```json
{ "id": "pit_fall",
  "steps": [
    {"prim": "set_prop", "args": ["$actor", "stunned_turns", 2]},
    {"prim": "emit",     "args": ["fell_into_pit", {"stun_turns": 2}]}
  ] }
```
- `steps` — упорядоченный список вызовов **ванильных Примитивов**. Имя вне
  `Primitives.PRIMITIVE_NAMES` → шаг отклонён (`mechanic_error`), Механика не
  может дотянуться до движка мимо алфавита.
- Крутит доверенный интерпретатор [mechanic_runner.gd](../../game/scripts/mechanic_runner.gd).
  Никакого сырого кода (сырой код — только сгенерированные Примитивы в песочнице, M4).

### Подстановка контекста

В `args` токен `"$имя"` подставляется из контекста запуска:
`$actor` (по умолчанию "player"), `$cell` (Vector2i события), `$target` (первый
не-игрок на клетке), `$data` (сырые данные события). Словари резолвятся рекурсивно.

## Скриптовый триггер

```json
{ "on": "player_moved",
  "if": {"cell_has_tag": "pit"},
  "mechanic": { ... } }
```
На событие `on`, если условие `if` истинно, интерпретатор запускает Механику —
мгновенно, в движке, ноль LLM. Условия M1: `{"cell_has_tag": <tag>}` (проверяет
объекты на клетке события). Условия расширяемы (Проверки, свойства — по мере нужды).

## Эскалация

```json
{ "id": "pit_frustration", "watch": "fell_into_pit",
  "threshold": 3, "emit": "pit_escalation" }
```
Детерминированный счётчик [trigger_system.gd](../../game/scripts/trigger_system.gd):
считает событие `watch`; на пороге запускает `mechanic` и/или эмитит `emit`, затем
сбрасывается. Так «важность» события решается кодом, а не Наблюдателем. На M4
`emit`-событие будит Режиссёра (Пробуждающий триггер).

## Защита

Интерпретатор enforce'ит вайтлист Примитивов; система триггеров имеет guard
рекурсии (`MAX_DEPTH`) на случай, если Механика эмитит в собственный триггер.
Всё — доверенные данные (M1); недоверенный сгенерированный код — тема песочницы M4.
