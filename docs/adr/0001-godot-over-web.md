# Godot 4.7, а не веб-стек

Рассматривались веб (TypeScript + PixiJS), Godot и Unity. Выбран Godot (решение
пользователя, 2026-07-09): полноценный движок с физикой и инструментами, при этом
две критичные для проекта возможности подтверждены документацией — рантайм-компиляция
GDScript (`Script.source_code` + `reload()`) и рантайм-рендер SVG
(`Image.load_svg_from_string`, Godot 4.7). Плата: ИИ-оркестрацию приходится выносить
в отдельный процесс (см. ADR-0002), автотестирование сложнее, чем в вебе (headless-режим).
