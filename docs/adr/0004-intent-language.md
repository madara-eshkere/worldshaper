# Режиссёр говорит Намерениями и никогда не пишет код

Альтернатива — позволить главной модели напрямую генерировать код изменений мира —
отклонена: дорого по токенам «личности», ненадёжно, неконтролируемо. Решение:
Режиссёр выражает изменения мира только декларативными Намерениями; Инженер
компилирует их по лестнице «композиция Примитивов → поиск в Библиотеке механик →
генерация нового кода (в последнюю очередь)». Новые Механики после валидации
сохраняются в Библиотеку навсегда — способности игры растут с каждой сессией, а
доля дорогой кодогенерации со временем падает.
