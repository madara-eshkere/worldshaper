# Контент: два профиля + несъёмный пол; фильтры чужих моделей не «отключаются»

Желание: полная настольная свобода («режим вообще без рамок») + безопасный режим
для сторов и стримеров, причём даже безопасный — без соплей (жестокость, чёрный
юмор, лёгкая эротика допустимы).

Реальность, от которой проектируем: у сторонних моделей отказы вшиты в веса
обучением — их нельзя «выключить тумблером», а попытки обхода промптом нарушают
ToS провайдеров и не могут быть фичей продукта. Поэтому:

1. **Контент-профили** (наша политика, версионируется с Персоной):
   **Standard** (~16+, дефолт: тёмное фэнтези, жестокость, чёрный юмор, лёгкая
   эротика — да; явная эротика — нет) и **Unrestricted** (18+, age-gate; полная
   настольная свобода — в пределах того, что позволяет выбранная модель).
2. **Несъёмный пол** в любом режиме: ничего незаконного, никаких реальных людей,
   никакого контента с несовершеннолетними. Не обсуждается ни этически, ни
   стор-политиками (Steam требует описания контроля AI-контента).
3. **Свобода через выбор модели, а не взлом**: разрешительность — свойство модели;
   BYOK-онбординг рекомендует модели под выбранный профиль (открытые/пермиссивные
   для Unrestricted; строгие модели честно помечаются как несовместимые с ним).
4. **Обработка отказов в сайдкаре**: отказ модели никогда не показывается игроку
   сырым — ретрай с переформулировкой в жанровых рамках или обыгрыш Персоной.
5. **В Standard наш Валидатор — второй контур** (чек-лист профиля поверх модели);
   в Unrestricted наш контур отключён, остаётся только пол из п.2.
6. **v2 (своя open-weights модель)**: базовая модель без лишней цензуры + наш
   контур как единственный фильтр — идеал пользователя достигается там.
