# Экономика релиза: разработка на фронтире, v1 — BYOK, v2 — своя открытая модель

Игра выпускается публично; платить за инференс всех игроков из кармана разработчика
нельзя, а продажа токенов внутри игры — главный источник негатива в вышедших
ИИ-играх (Suck Up!, Chongzhen Simulator). Решение: (1) разработка — фронтирные
модели через Адаптер LLM; (2) релиз v1 — BYOK по модели Mantella: ключ OpenRouter
или локальная модель игрока (Ollama/LM Studio); (3) при успехе v2 — открытая модель
(open-weights) на своём/арендованном инференсе + подписка или сессии в цене игры,
позже возможна дистилляция на логах. Следствие: ни один компонент не должен зависеть
от конкретного провайдера; качество должно быть приемлемым на «полу» из дешёвых
открытых моделей (DeepSeek/Qwen-класс, включая русский язык).
