# World State — реестр-JSON; Примитивы — единственный мутатор и граница песочницы

Механикам (в т.ч. сгенерированным ИИ, ADR-0006) нужен способ менять мир, который
(1) безопасен для песочницы, (2) сериализуем для ИИ и сейвов, (3) не даёт гостю
трогать движок напрямую. Рассматривалось: объекты = Godot-ноды как источник правды
(гостю пришлось бы давать доступ к нодам — та самая дыра, которую мы закрываем).

Решение: **World State = реестр объектов (Dictionary id→{type,cell,props,tags}),
сериализуемый в JSON — единственный источник правды**; Godot-ноды это лишь
представление, синхронизируемое по сигналам. Всякая запись в мир идёт **только**
через фиксированный алфавит **Примитивов** (docs/specs/primitives.md), которые
валидируют аргументы и не роняют движок на мусоре. Вайтлист песочницы = объект
Primitives + его методы; больше гостю не доступно ничего.

Следствия: сейвы и «мир для ИИ» — это сериализация реестра (совпадает с DESIGN.md
4.7); рендеринг отделён от логики (тестируется headless без графики); лестница
Инженера (ADR-0004) компонует Примитивы; добавление Примитива — единственный способ
расширить возможности Механик, что держит поверхность атаки под контролем.
